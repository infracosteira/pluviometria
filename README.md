# InfraCosteira — Plataforma de Dados Pluviométricos 🌧️

### O InfraCosteira é uma aplicação web desenvolvida para visualização, consulta e download de dados pluviométricos de estações meteorológicas do estado do Ceará. A plataforma organiza informações históricas de forma acessível, permitindo análises rápidas por município e posto de coleta.

## 🔍 Funcionalidades
### 🔎 Busca por município
* Permite localizar rapidamente estações pluviométricas específicas.
### 📊 Tabela interativa
* Dados organizados com suporte a ordenação por colunas (ID, posto, município, período, etc.).
### ⬇️ Download de dados
* Cada estação possui acesso direto aos seus dados completos.
### 🌍 Visualização geográfica
Integração com mapa para localização dos postos.
### 📅 Cobertura temporal
* Exibição do período disponível (ano inicial, final e total de anos completos).
